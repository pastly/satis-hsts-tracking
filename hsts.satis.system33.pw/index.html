HSTS SATIS Demo

No state is ever kept on the server, and once you allow us, no state will be kept in your browser for more than 30 minutes after your last visit.

--> Click here <-- if you would like us to start tracking the last time you visited this website. This demo works without JavaScript and primarily demonstrates we can give different content to users based only on the contents of their HSTS state.

Are you using Chrome? You can view your HSTS state information! See here.

Are you using Safari? You can view your HSTS information using a script that parses this file.

Are you using Firefox? You can view your HSTS information by opening SiteSecurityServiceState.txt in your profile directory. Firefox seems to keep HSTS information in memory and only update this file when closing.

Are you using Tor Browser? Tor Browser also keeps HSTS information in memory and we found a SiteSecurityServiceState.txt file in its profile directory. However, it never seemed to update the file and after deleting the file, we couldn't convince Tor Browser to recreate it. Search for HSTS in Tor Browser's design document for information on what they have done to protect against HSTS attacks.
